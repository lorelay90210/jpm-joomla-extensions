Availability Calendar

De Availability Calender plaatst in een artikel d.m.v. een kalender weergave de vrij/bezet status van een object. Deze extensie bestaat uit een "administration" component com_availcal en een plugin plg_availcal. Via de component worden de bezette periodes (dark periods) ingevuld.
Input:
Naam de naam van het object
Startdatum de eerste dag van de "bezette" periode (jjjj-mm-dd)
Einddatum de laatste dag van de "bezette" periode (jjjj-mm-dd)
Bezet deels of geheel. Hiermee kan aangegeven worden of het object beperkt of niet beschikbaar is in de periode
Opmerking optioneel en wordt alleen getoonde in de administratie component

Door {availcal="naam"} in een artikel te plaatsen wordt de "bezet" status getoond.
D.m.v. de plugin configuratie wordt het aantal getoonde maanden (vanaf de huidige maand) bepaald.

Invoer van Darkperiods d.m.v. een XML file

Voor synchronisatie met andere applicaties is een functie beschikbaar om een XML file in te lezen, UploadXML.

De root node van het XML document is "objects".
Vervolgens is er de "object" node (kind node), met als attribuut de naam van het object.
De object node omsluit 4 andere nodes startdate, enddate, busy en optioneel remark. Startdate en enddate hebben het formaat yyyy/mm/dd.
Voor busy is de waarde 0 gedeeltelijk bezet, de waarde 1 voor volledig bezet en de waarde 2 voor niet bezet.
Deze laatste waarde is er voor die applicaties die alleen een volledig kalender in XML formaat kunnen genereren.
Availcal slaat de periodes met een busy waarde 2 over.

Het bestand availcal_02.zip bevat een voorbeeld XML file anderespanje.xml en een XMLSchema availcalXMLschema.xsd.
